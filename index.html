GollumCoreLib

It's library for Minecraft Mod. Add functionalities :

  • Blocks Factory
  • Items Factory
  • SoundRegistery
  • Building builder
  • Logger
  • Version Checker
  • Block entity auto-spawner
  • Mod metadatas getter
  • Annotation config loader
  • Several Helper
Gollumcorelib is maintained by GollumTeam
